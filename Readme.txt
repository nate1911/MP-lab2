
Endpoints
List of all available endpoints with a description of each one.

Endpoint 1
- Description: This endpoint posts a new recipe into the database.
- Method: POST
- URL: http://localhost:3000/api/recipes

Endpoint 2
- Description: This endpoint gets all recipes from the database.
- Method: GET
- URL: http://localhost:3000/api/recipes

Endpoint 3
- Description: This endpoint gets specific recipe from the database using title.
- Method: GET
- URL: http://localhost:3000/api/recipes/:title

Endpoint 4
- Description: This endpoint gets deletes specific recipe from the database.
- Method: DELETE
- URL: http://localhost:3000/api/recipes/:id

Endpoint 5
- Description: This endpoint updates specific recipe from the database.
- Method: PUT
- URL: http://localhost:3000/api/recipes/:id
